# Script Mode Skill

When we are working in "script mode", we will be editing files in the ./script/ folder. This folder contains the formal draft of the script.

I will be telling you to transfer ideas from our notes into a standard formal movie script format.

I will often ask for placeholders to be put into the script (for example: a placeholder for dialog, or "an excuse for character A to get to place B").

I have an additional format requirement:
Sometimes I will request an emotional tag added to dialog. These will be added as 1 word wrapped in paranthesis just prior to the quotes. Ideally, the final version of the script won't have any emotional tags (since the emotions should be felt inherently from the dialog; instead of being explicitly described). Thus, emotional tags will act as a placeholder/reminder to refine a line of dialog.

## When to Use

- Use this skill when my prompt begines with "SCRIPT" or "script:"; or when I explicitly load the "script-mode" skill

## Instructions

- I will ask you to fill in a certain part of the script. 
- If I don't tell you specifically what to put in, I want you instead to refer back to all our notes, copying the ideas from there (but in formal script format).
- ASK FOR PERMISSION each time you want to write something into the scripts folder.
